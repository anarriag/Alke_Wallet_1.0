## Herramientas de Software utilizadas.

- Lenguaje de marcado HTML.
- Lenguaje de estilos CSS.
- Lenguaje de Programación JavaScript.
- Bootstrap.
- JQuery.

## Objetivo a desarrollar.

Construir una aplicación de billetera digital que permita a los usuarios gestionar sus activos financieros de manera segura y conveniente mediante el uso de las herramientas de software antes indicadas con visualización responsive.

## Conclusiones finales.

Un proyecto Web, debe ser modular y estructurado en lo concerniente a los diferentes archivos que lo componen. Esto es con el objetivo de que sea mantenible en el tiempo asi como tener la posibilidad de reutilizar código y de agregar nuevos componentes sin tener que reestructurar de manera significativa el código HTML.

Con el aprendizaje de nuevas herramientas de desarrollo, es muy posible que el código JavaScript, CSS y HTML del presente proyecto, se optimice a futuro haciendolo más eficiente.
